Software & Tools:

Although you are not required to use the Zotero - Better BibTeX - citr workflow, I strongly recommend at least trying it. Follow the class guide to set it up, or use just the elements you find most useful.

  • Zotero (free citation management)
  • Better BibTeX (Zotero extension)
  • citr (RStudio package/addin)
    • Note: This package is not on CRAN and cannot be installed with install.packages(). Use devtools::install_github("crsh/citr") instead.
  • CSL Project (many, many pre-built .csl files for common and uncommon citation styles)
    • The github with the actual files
    • Note: apaquarto will use its own embedded APA7 .csl by default. You can override this or edit the file within the extension, but I don’t recommend it unless you know exactly why you need to and how to do so.
